# 为什么使用Python做数据分析？

1. 数据分析相关的核心库又多又好用；
2. 外围库支持又多又好用，工程化容易；（写爬虫，写Web应用，各种数据处理包，各种数据库接口）
3. 强大的胶水性能，统一所有分析工具的优点；（调用Julia、R、Stata、Matlab等）
4. 好学好写，语法简单优雅，可读性强。
